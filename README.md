# House-floor-designs

Wall dimentions via [dimensionlines](https://sourceforge.net/projects/dimensionlines/) plugin (for Windows, move it to _C:\Users\"Your user name"\AppData\Roaming\eTeks\Sweet Home 3D\plugins_)
